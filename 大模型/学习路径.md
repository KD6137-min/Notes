

-   **CoT (Chain-of-Thought，思维链)**
    -   **它是什么**：一种提示技术。让模型在给出最终答案前，先输出一步步推理过程，极大提高复杂推理的正确率。
    -   **你看到的代码**：在提示词里加上 `Let's think step by step`，或者处理模型输出时，把推理步骤和最终答案分开解析。
    -   **核心本质**：改善模型“思考”方式的**提示工程**。

-   **Agent (智能体)**
    -   **它是什么**：给LLM装上“手”和“眼”，让它能调用工具、执行计划、与环境交互，成为一个自主完成任务的实体。比如，一个能帮你查机票、写邮件、订酒店的AI。
    -   **你看到的代码**：`import openai` 只是第一步，后面会接上定义好的工具函数（如 `search_flights`），然后写出一个循环：LLM思考下一步动作 → 调用工具 → 获取结果 → 再思考，直到任务完成。
    -   **核心本质**：一套**让LLM动态决策并行动**的控制流架构。

-   **Harness (套具/框架)**
    -   **它是什么**：为了快速开发、测试、部署上述Agent，而出现的集成框架。它封装了提示管理、工具调用、记忆等模块。
    -   **你看到的代码**：`from langchain.agents import ...` 或 `import autogen`，就是导入了一个现成的“马具”，帮你把LLM这匹“马”驾驭起来。
    -   **核心本质**：工程化的**脚手架和最佳实践集合**。





1.  **大语言模型（LLM）的核心机制**
    -   **预训练**：知道语言模型的目标是“预测下一个词”，海量数据下涌现出了能力。
    -   **指令微调**：让模型从“补全”模式，变成“听懂指令并回答”的助手模式。
    -   **RLHF**：基于人类反馈的强化学习，让模型价值观与人类对齐。这是ChatGPT成功的关键。

2.  **提示工程与思维链**
    -   **轻**：背各种提示模板。
    -   **重**：理解为什么CoT有效。它是让模型在“工作记忆”里展开中间草稿，将复杂问题分解。你自己用API做实验，对比下加了 `Let's think step by step` 和没加的输出，就全明白了。

3.  **LLM API调用与微调**
    -   **轻**：`import openai`，调个接口。
    -   **重**：理解“**上下文学习**”。你没动模型参数，只是在提示词里给了几个范例，模型就学会了新任务。这是研究前沿，也是Agent的基础。学有余力，去Google Colab上跑一次LoRA微调开源模型（如Llama），感受全过程。
4.  **Agent框架**
    -   **学什么**：搞懂Agent的核心循环：**思考(Thought) → 行动(Action) → 观察(Observation)**。理解了这点，你就不会被LangChain、AutoGen等不同框架的表象迷惑。**建议直接学原生的OpenAI Assistants API或Function Calling来手写一个小Agent，这比用任何框架都更触及本质。**

5.  **LangChain, LlamaIndex 等Harness**
    -   **态度**：它们是工具，不是学问。当你对上述所有原理都有认识后，你需要快速开发一个RAG应用，去花一天看文档就足够了。现在不需要花时间学。

-   **第1-2周：补基础与原理**
    -   目标：搞懂Transformer。
    -   资源：李沐《动手学深度学习》Transformer章节 + 哈佛大学《The Annotated Transformer》博客。不要只看，要跟着代码一行行实现。

-   **第3-4周：理解LLM与上手实验**
    -   目标：搞懂ChatGPT是怎么炼成的。
    -   资源：李宏毅2024年《生成式AI导论》课程前几讲。
    -   **关键动作**：申请一个OpenAI/国产大模型API Key，在Jupyter Notebook里调用。分别实验Zero-shot、Few-shot、CoT的效果，记录感受。这就是你对CoT最直观的认知。

-   **第5-6周：实现一个最简单的Agent**
    -   目标：脱离框架，写一个最原始的智能体。
    -   **关键动作**：不借助LangChain，自己写一个Python循环。定义一个工具函数，比如 `def calculator(expression)`，让LLM输出一个固定格式来调用它。你会发现，几行代码就能完成这个循环，瞬间理解Agent本质。

-   **第7周及以后：自由探索**
    -   此时，你可以快速浏览LangChain文档，做个小项目；或者去读进阶论文，比如ReAct、AutoGPT。因为有了坚实基础，你看到的不再是魔法，而是巧妙的设计。
