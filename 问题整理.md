# torch安装雷

- conda install pytorch区别于pip install torch，不要混用，conda安装pytorch必须安装gpu版本的，若错装，必须删除后重装，否则系统仍默认使用cpu
- PyTorch 团队没有在 `pytorch` 频道里提供 CUDA 版本给 Windows Conda，所以不要用`conda install pytorch torchvision -c pytorch`
- 不推荐用uv pip，易导致cuda不可用/版本冲突
- conda-forge 的 openssl 3.6.0 + 超前的 ca-certificates（2025 年版）高概率破坏 pip SSL，导致连不上任何https服务器
- Windows + Python3.8 + openssl3.x = 容易爆 SSL 问题
- PyTorch 2.4 官方 **已经停止支持 Python 3.8**
- 清华 pypi 源（simple 源）只能提供 CPU 版本的 PyTorch，GPU 版本只能从清华的pytorch/wheels专用目录安装



最终：

```python
# env
conda create -n pytorch-gpu python=3.10
conda activate pytorch-gpu

# install
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

# validate
python -c "import torch; print(torch.__version__); print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0))"
```



**放弃使用miniforge和uv！**
